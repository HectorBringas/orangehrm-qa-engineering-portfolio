# OrangeHRM QA Engineering Portfolio

OrangeHRM QA Engineering Portfolio
Overview

This repository demonstrates a complete QA Engineering approach using OrangeHRM as the system under test.

The goal of this project is to simulate a real-world QA environment and showcase Senior QA competencies, including:

Quality Strategy
Risk-Based Testing
Test Design
UI Automation
API Testing
Database Validation
CI/CD Integration
Reporting and Metrics

The project is developed as a portfolio initiative to demonstrate both technical and quality engineering decision-making.

Project Objectives
Design a professional QA strategy.
Identify and mitigate product risks.
Build a scalable automation framework.
Validate functionality across UI, API, and database layers.
Implement CI/CD quality gates.
Produce quality metrics and reporting.
